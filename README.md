# 🎨 Color Palette Generator

A responsive web app that lets users generate stunning color palettes using **Single**, **Complementary**, and **Triadic** schemes. Built with vanilla **JavaScript**, **HTML**, and **CSS**, this tool is perfect for designers, developers, and creatives looking to explore dynamic color combinations.

---

## 🚀 Features

- 🔹 Generate palettes in **RGB** or **HEX** format
- 🔹 Choose between **Single**, **Complementary**, or **Triadic** color schemes
- 🔹 Responsive layout for desktop and mobile
- 🔹 Copy color codes with one click
- 🔹 Clean UI with gradient headers and dark theme

---

## 📱 Mobile Optimization

- Palette boxes resize and stack vertically on smaller screens
- Dropdowns and buttons adapt for touch-friendly interaction
- Layout remains clean and readable across devices

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Custom properties, Flexbox, Grid, Media Queries)
- JavaScript (DOM manipulation, color logic)

---

---
## 💡 How to Use

1. Clone the repo:
   ```bash
   git clone https://github.com/your-username/color-palette-generator.git
   ```

2. Open `index.html` in your browser

3. Select format (RGB or HEX) and palette type

4. Click **Generate Colors** to explore combinations
